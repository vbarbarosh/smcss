## npm scripts / bin

```
npm run build   rm -rf build
                sass demos/demo.sass | postcss -o demos/demo.css     (demo page styles)
                sass demos/sm.sass   | postcss -o dist/sm.css        (the library)
                cp reset/reset.css dist/reset.css
npm test        mocha (test/index.js) — compiles expressions with dart-sass, compares csso-minified output
bin/build       wrapper around npm run build
bin/test        wrapper around npm test
bin/configure   npm install
bin/update-docs bin/build → regenerate docs/ from src/docs/*.php (needs PHP)
bin/release     major|minor|patch: test → npm version --no-git-tag-version →
                update-docs → git add package.json package-lock.json docs demos dist →
                commit "release vX.Y.Z" → tag → push → npm publish
```

- postcss runs autoprefixer only (`postcss.config.js`), browserslist defaults.
- **`dist/` is committed** and rebuilt at release time; `docs/` likewise.

## What gets published

`package.json files`: `src/smcss` + `dist/*.css`. Entry points:
- `main: index.sass` (SASS consumers)
- `unpkg`/`jsdelivr: dist/sm.css` (browser `<link>`)
- `dist/sm.css` is ~332 KB / 38k lines (unminified, no `.min.css` variant) —
  driven by brute-force enumeration `w0..w1200`, `h0..h1200`, `lh0..lh100`,
  `br1p..br100p`, `min-/max-` 0..500, etc. (generated with bash
  `echo w{0..1200}` and pasted into demos/sm.sass).

## Docs site (smcss.vbarbarosh.com)

- Static HTML in `docs/` (GitHub Pages, CNAME committed), generated by PHP
  templates in `src/docs/`: `concepts/` (container, hsplit, margin-group),
  `reference/` (one PHP file per module — the class-by-class reference),
  `demos/` (index over copied demos), `try/` (live playground).
- `src/docs/php/app.php` provides `e()`, `version()` (reads package.json),
  `snippet()` (extracts `<body>` of a demo file, de-indents) and the CodePen
  export helper (recent commits: "add smcss link to CodePen exports").
- `try/` playground: CodeMirror 5 from cdnjs, resizable editor/preview split,
  "prefill from a demo…" dropdown built by globbing `demos/*/ *.html` at
  generation time; light/dark theme via `localStorage.smcss_theme`.
- Local preview of PHP pages: `php -S localhost:8000` from repo root.
